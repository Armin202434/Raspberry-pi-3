Was kostet der Raspberry Pi 3?

Der Raspberry Pi 3 kostet etwa 35-50€, abhängig vom Händler und Zubehör. Eine großartige Wahl für Anfänger und Enthusiasten!

Spezifikationen

Technische Daten

  • 1.2 GHz Quad-Core ARM Cortex-A53 CPU
  • 1 GB RAM
  • Integriertes WLAN und Bluetooth
  • 4 USB-Ports, HDMI, und 40 GPIO-Pins
  • MicroSD-Kartenslot für das Betriebssystem
Tutorial

Website mit Visual Studio Code und GitHub Pages

  1. Erstelle deine Website in **Visual Studio Code**:
    • Erstelle eine neue Datei `index.html` und füge den HTML-Code ein.
    • Optional: Erstelle separate Dateien für CSS und JavaScript.
  2. Teste die Website lokal:
    • Installiere die Erweiterung **Live Server** in Visual Studio Code.
    • Öffne deine Datei mit Live Server, um die Änderungen in Echtzeit zu sehen.
  3. Lade die Website auf GitHub hoch:
    • Erstelle ein neues GitHub-Repository und füge deine Dateien hinzu.
    • Nutze folgende Befehle im Terminal: 
      git init
git add .
git commit -m "Website initial commit"
git branch -M main
git remote add origin https://github.com//.git
git push -u origin main
  4. Aktiviere GitHub Pages:
    • Gehe in deinem Repository zu **Settings → Pages**.
    • Wähle den Branch `main` und klicke auf **Save**.
  5. Öffne die Website auf dem Raspberry Pi:
    • Starte den Webbrowser auf deinem Raspberry Pi.
    • Gib die GitHub Pages URL ein (z. B. `https://username.github.io/repository-name`).
    • Die Website ist nun auf dem Raspberry Pi sichtbar!
