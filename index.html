Was kostet der Raspberry Pi 3?

Der Raspberry Pi 3 kostet etwa 38-55 CHF, abhängig vom Händler und Zubehör. Eine großartige Wahl für Anfänger und Enthusiasten!

Spezifikationen

Technische Daten

  • 1.2 GHz Quad-Core ARM Cortex-A53 CPU
  • 1 GB RAM
  • Integriertes WLAN und Bluetooth
  • 4 USB-Ports, HDMI, und 40 GPIO-Pins
  • MicroSD-Kartenslot für das Betriebssystem
Tutorial

Konfiguration des Raspberry Pi 3

  1. Installiere ein Betriebssystem wie Raspberry Pi OS auf einer MicroSD-Karte.
  2. Starte den Raspberry Pi und aktiviere SSH unter den Einstellungen.
  3. Verbinde dich mit einem SSH-Client wie PuTTY oder Terminal (macOS/Linux) mit dem Raspberry Pi.
  4. Installiere den RealVNC-Server auf dem Raspberry Pi mit sudo apt install realvnc-vnc-server.
  5. Starte den VNC-Server und richte ihn unter den Einstellungen ein.
  6. Installiere den RealVNC-Viewer auf deinem PC oder Mobilgerät.
  7. Verbinde dich mit dem Raspberry Pi, indem du die IP-Adresse eingibst und die Anmeldedaten verwendest.
  8. Jetzt kannst du den Raspberry Pi aus der Ferne steuern!

Für weitere Details kannst du das Tutorial als PDF ansehen: Raspberry Pi 3 Konfigurieren Tutorial.
